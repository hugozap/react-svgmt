## [1.1.3]

Added 2 new components to facilitate transform animation:

- TransformMotion
- Attribute Motion

```
<TransformMotion
            selector="#Group"
            target={{
              x: target.x,
              y: target.y,
              angle: target.angle,
              rotateX: 183,
              rotateY: 177
            }}
          />
          <AttributeMotion
            selector="#Group"
            start={{ opacity: 0 }}
            target={{ opacity: opacity }}
          />
```
